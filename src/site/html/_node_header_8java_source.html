15  /*
16  The Header of the block when stored as bytes is:
17  (1) Byte - Is this a leaf block?
18  (4) int - The size of the keys, only relevant if keys are the same length.
19  (4) int - the number of keys in this node. A slightly delicate topic in the internal nodes, since they also contain child node ids and not just keys.
20  (8) long - the id to the next node, the sibling node.
21  (8) long - the id to the previous node, the preceding node.
22  */
