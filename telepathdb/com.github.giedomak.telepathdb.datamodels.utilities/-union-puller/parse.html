parse

fun parse(parseTree: ParseTree): List<ParseTree> (source)

Removes the UNION operatorId from a ParseTree.

Simply duplicate the tree and on the position of the UNION operatorId, the first tree gets the left-child and the other tree gets the right-child.

Parameters

parseTree - The ParseTree to parse.

Return
List of ParseTrees each not containing the UNION operatorId.
