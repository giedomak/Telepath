fun parse(logicalPlan: LogicalPlan): List<LogicalPlan> (source)

Removes the UNION operator from a LogicalPlan.

Simply duplicate the tree and on the position of the UNION operator, the first tree gets the left-child and the other tree gets the right-child.

Parameters

logicalPlan - The LogicalPlan to parse.

Return
List of ParseTrees each not containing the UNION operator.
