fun createLookupTree(pathId: Long): ParseTree (source)

Creates a LOOKUP parseTree given a pathId.

This function creates a parseTree where the root has the operator LOOKUP and where it has only one level of children. Each edge along the pathId will be a leaf child from this root.

Parameters

pathId - The ID indicating a path from which we will get the list of edges.

Return
A ParseTree with depth 2, where the root is the operator LOOKUP and its children are leafs with the edge labels from the pathId.
